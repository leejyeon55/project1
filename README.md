## 프로젝트 제작 동기
- 디자인부터 코딩까지 100% 혼자 작업한 프로젝트입니다.

- html5, css3, 바닐라 자바스크립트를 사용해서 구현한 웹 사이트입니다.
- 반응형 페이지로 구현 하였습니다.
